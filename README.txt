To run the project:
1.Command "python aosipa_a.py" for part A or basic TTT game. All the other required input will be prompted.
2.Command "python aosipa_b.py" for part B or 9 board TTT game. All the other required input will be prompted.
Both commands will run the game itself immediately.

To compile:
1.Command "python -m py_compile aosipa_{a,b}.py".
